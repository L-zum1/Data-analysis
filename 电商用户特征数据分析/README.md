# 电商用户特征数据分析

## 项目简介
该项目是一个基于Python的电商用户特征数据分析工具，通过对用户个性化特征数据的深入分析，帮助企业了解用户行为模式、消费习惯和价值分层，为精准营销和用户运营提供数据支持。

## 功能特点

### 1. 数据处理与清洗
- 数据格式转换与标准化
- 中文列名与值映射
- 缺失值处理

### 2. 用户分层分析
- **年龄群体划分**：少年、青年、中年、中老年、老年
- **收入群体划分**：低收入、中低收入、中高收入、高收入
- **RFM模型分析**：基于最近登录时间(R)、消费金额(M)、购买频率(F)的用户价值分层

### 3. 多维度用户特征分析
- 性别分布分析
- 年龄群体与性别交叉分析
- 位置分布分析
- 兴趣分布分析
- 订阅活动通知对用户行为的影响

### 4. 消费行为分析
- 整体客单价计算
- 订阅用户与未订阅用户客单价对比
- 购买频率与消费金额关系分析

### 5. 数据可视化
- 饼图：性别分布、兴趣分布
- 柱状图：位置分布、年龄群体性别分布
- 交叉分析图表

## 技术栈
- Python 3.7+
- pandas：数据处理与分析
- numpy：数值计算
- matplotlib：数据可视化
- seaborn：高级数据可视化

## 项目结构
```
电商用户特征数据分析/
├── main.py              # 主程序入口，包含所有分析和可视化功能
└── user_personalized_features.csv  # 用户特征数据文件
```

## 数据说明

### 数据字段
- **用户ID**：用户唯一标识
- **年龄**：用户年龄
- **性别**：用户性别（男/女）
- **位置**：用户所在地区（城市/郊区/农村）
- **收入**：用户收入水平
- **兴趣**：用户兴趣标签（运动/科技/时尚/旅行/美食）
- **最近登录时间**：距离最近一次登录的天数
- **购买频率**：用户购买频率
- **平均消费金额**：用户平均每次消费金额
- **总消费金额**：用户总消费金额
- **产品类别偏好**：用户偏好的产品类别
- **平台使用时间**：用户在平台的使用时间（分钟）
- **访问页面数**：用户访问的页面数量
- **是否订阅活动通知**：用户是否订阅活动通知（是/否）

## 安装与运行

### 1. 环境准备
确保已安装Python 3.7或更高版本

### 2. 安装依赖
```bash
pip install pandas numpy matplotlib seaborn
```

### 3. 运行程序
```bash
# 进入项目目录
cd 电商用户特征数据分析

# 运行主程序
python main.py
```
