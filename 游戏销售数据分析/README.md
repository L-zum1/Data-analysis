# 游戏销售数据分析项目

## 项目简介
本项目是一个基于Python的游戏销售数据分析工具，用于分析视频游戏销售数据并进行可视化展示。

## 功能特性
1. **数据处理**：对游戏销售数据进行清洗和整理
2. **数据分析**：
   - 分析不同游戏类型的全球销售数据
   - 分析不同游戏平台的销售表现
   - 分析每年游戏发行数量趋势
   - 分析各地区游戏销售数据趋势
   - 分析游戏销售数据相关系数
   - 分析主要游戏发行商的销售表现
3. **可视化展示**：
   - 所有年份游戏类型销售数据柱状图
   - 最近5年游戏类型销售数据柱状图
   - 所有年份游戏平台销售数据柱状图
   - 最近5年游戏平台销售数据柱状图
   - 每年游戏发行数量柱状图
   - 各地区游戏销售数据折线图
   - 游戏销售数据相关系数热力图
   - 主要游戏发行商销售数据折线图
   - 游戏类型和发行商销售数据堆叠柱状图

## 技术栈
- **数据处理**：pandas, numpy
- **数据可视化**：matplotlib, seaborn
- **其他**：os

## 使用方法

### 1. 准备数据
下载游戏销售数据文件 `vgsales.csv` 并保存到适当位置。
数据来源：VGChartz 游戏销售数据库

### 2. 安装依赖
```bash
pip install pandas matplotlib seaborn numpy
```

### 3. 运行分析脚本
修改 `game.py` 中的数据文件路径：
```python
df = pd.read_csv('your_path_to/vgsales.csv')
```

然后运行脚本：
```bash
python game.py
```

### 4. 查看结果
脚本运行后会在当前目录下生成多个可视化图表文件：
- 游戏类型销售数据.png
- 游戏平台销售数据.png
- 每年游戏发行数量.png
- 各地区游戏销售数据.png
- 游戏销售数据相关系数热力图.png
- 游戏销售数据，游戏发行年份，游戏发行商排名可视化.png
- 游戏类型和发行商销售数据.png

## 数据说明
数据包含以下主要字段：
- Rank: 游戏排名
- Name: 游戏名称
- Platform: 游戏平台
- Year: 发行年份
- Genre: 游戏类型
- Publisher: 发行商
- NA_Sales: 北美销售额（百万美元）
- EU_Sales: 欧洲销售额（百万美元）
- JP_Sales: 日本销售额（百万美元）
- Other_Sales: 其他地区销售额（百万美元）
- Global_Sales: 全球销售额（百万美元）
